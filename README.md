## ⚙️ Installation

**Python de base (cours + TPs Python) :**
```bash
git clone https://github.com/FantoniStephane/formation-cd2ia.git
cd formation-cd2ia

python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux

pip install -r requirements.txt
```

**Machine Learning (TPs ML) :**
```bash
python -m venv venv_tf
venv_tf\Scripts\activate     # Windows
source venv_tf/bin/activate  # Mac/Linux

pip install -r requirements_tf.txt
```
