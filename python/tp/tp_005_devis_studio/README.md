# TP 005 — Calculateur de devis pour un studio de création

**Notions abordées :** fonctions, `def`, `return`, dictionnaire, `bool`  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Calcule le prix estimé d'un devis selon le type de prestation, le nombre d'heures, l'urgence et la fidélité du client.

---

## Exemple de déroulé

```
Bienvenue dans le calculateur de devis.
Type de prestation (logo/site/affiche) : logo
Nombre d'heures : 5
Livraison urgente ? (oui/non) : oui
Client fidèle ? (oui/non) : non

Récapitulatif du devis :
Prestation : logo
Nombre d'heures : 5
Prix final estimé : 455.0 €
```
