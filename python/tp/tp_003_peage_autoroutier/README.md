# TP 003 — Le péage autoroutier

**Notions abordées :** dictionnaire, `while`, `if / elif / else`, `int()`, `max()`  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Simule un péage autoroutier. Calcule le tarif selon le type de véhicule, le nombre de passagers et la présence d'un abonnement.

---

## Exemple de déroulé

```
Bienvenue au péage automatique.
Type de véhicule : voiture
Nombre de passagers : 4
Abonnement ? (o/n) : o
Tarif initial : 5 €
Réduction covoiturage : 1 €
Réduction abonnement : 2 €
Tarif final : 2 €
Passage autorisé.
```
