# TP 001 — Jeu "Devine le nombre"

**Notions abordées :** `random`, `while`, `if / elif / else`, `input()`, `int()`  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Un jeu en terminal où le programme choisit un nombre secret entre 1 et 100.  
Le joueur doit le deviner — après chaque essai, le programme indique si le nombre est plus grand ou plus petit.

---

## Exemple de déroulé

```
Bienvenue dans le jeu Devine le nombre.
Essayez de trouver le nombre secret.

Votre essai : 20
Le nombre est plus grand !

Votre essai : 70
Le nombre est plus petit !

Votre essai : 42
Bravo ! Vous avez bien deviné 42
```
