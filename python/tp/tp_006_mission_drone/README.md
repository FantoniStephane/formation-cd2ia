# TP 006 — Simulateur de mission de drone d'inspection

**Notions abordées :** fonctions, modules, `while`, `if / elif / else`, `try/except`  
**Difficulté :** Intermédiaire

---

## Ce que fait ce programme

Simule la préparation d'une mission de drone. Vérifie les conditions de décollage, calcule un score de risque et affiche une décision finale.

---

## Exemple de déroulé

```
Bienvenue dans le système de préparation des missions de drone.
Nom de la mission : Inspection A17
Batterie du drone : 28
Météo : vent
Zone : toiture
Priorité : haute
Autorisation spéciale : non

Rapport de mission
Mission : Inspection A17
Batterie : 28 %
Score de risque : 7
Niveau de risque : élevé
Décision : Mission autorisée avec vigilance renforcée
```
