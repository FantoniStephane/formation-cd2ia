# TP 004 — Le bulletin scolaire

**Notions abordées :** fonctions, `def`, `return`, `float()`, `try/except`  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Calcule les moyennes par matière et la moyenne générale d'un élève, puis affiche la mention correspondante.

---

## Exemple de déroulé

```
Bienvenue dans le simulateur de bulletin scolaire.

Saisir les 3 notes de mathématiques :
Note 1 : 12
Note 2 : 15
Note 3 : 9

Bilan de l'élève :
Moyenne de mathématiques : 12.00
Moyenne de français : 12.67
Moyenne d'anglais : 16.00
Moyenne générale : 13.56
Mention : Assez bien
```
