# TP 009 — Générateur de signatures d'email

**Notions abordées :** fonctions, paramètres, valeurs par défaut, `return`, f-strings  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Génère automatiquement une signature d'e-mail professionnelle à partir des informations saisies. Si un champ est laissé vide, une valeur par défaut est appliquée.

---

## Exemple de déroulé

```
=== Générateur de signature e-mail ===

Nom : Martin
Prénom : Elise
Poste : Responsable RH
Téléphone : 06 12 34 56 78
Adresse e-mail :

--- SIGNATURE GÉNÉRÉE ---
Elise MARTIN
Responsable Rh
Tél : 06 12 34 56 78
Email : elise.martin@entreprise.com
```
