# TP 002 — Le coffre-fort mystérieux

**Notions abordées :** `input()`, `int()`, `if / elif / else`, `and`, variables  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Simule un système de sécurité pour ouvrir un coffre-fort.  
Le programme demande le prénom, le code secret et l'âge, puis détermine si l'accès est autorisé.

---

## Exemple de déroulé

```
Bienvenue dans le système de sécurité.
Quel est votre prénom ? John
Entrez le code secret : PY123
Quel âge avez-vous ? 45
Accès autorisé. Bienvenue John. Le coffre est maintenant ouvert.
```
