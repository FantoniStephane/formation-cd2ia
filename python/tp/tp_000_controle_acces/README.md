# TP 000 — Contrôle d'accès à une zone sécurisée

**Notions abordées :** `input()`, `int()`, conditions `if / elif / else`, `and`, `or`  
**Difficulté :** Débutant

---

## Ce que fait ce programme

Simule un système de contrôle d'accès à une zone sécurisée.  
Le programme demande 4 informations à l'utilisateur et détermine si l'accès est autorisé ou refusé.

---

## Exemple de déroulé

```
Entrez votre âge : 28
Avez-vous un badge valide (o/n) ? : o
Connaissez-vous le code du jour (o/n) ? : o
Êtes-vous accompagné d'un responsable (o/n) ? : n
Accès autorisé : accès complet validé.
```
