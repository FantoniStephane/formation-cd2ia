# TP 007 — Générateur de participants fictifs pour une conférence

**Notions abordées :** `requests`, API REST, JSON, fonctions, `try/except`  
**Difficulté :** Intermédiaire

---

## Ce que fait ce programme

Génère des profils fictifs via l'API Random User Generator et les affiche sous forme de badges participants.

---

## Exemple de déroulé

```
Bienvenue dans le générateur de participants fictifs.

Combien de participants voulez-vous générer ? : 3

Récupération des participants...

--- Participant 1 ---
Nom complet : Emma Martin
Email       : emma.martin@example.com
Ville       : Paris
Pays        : France
Nationalité : FR
```
